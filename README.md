# Plataforma ONG – Front-end

Projeto da disciplina de Front-end: plataforma web para ONGs com páginas de:
- Início (index.html)
- Projetos (projetos.html)
- Cadastro (cadastro.html)

Tecnologias: HTML5 semântico + CSS3 modular (design system, grid, flex, responsivo) + JS básico.

Estrutura de pastas:
- assets/css → estilos em módulos
- assets/js → scripts (mobile menu, ano do rodapé)
- assets/images → imagens usadas no layout
